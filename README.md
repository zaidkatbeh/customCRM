
# CRM

an admin panel to manage clients,users,projects and tasks



## Features

- view all clients
- view all users
- view projects
- view tasks
- add projects
- add tasks
- manage projects(view,delete and update)
- manage users (view,delete and update)
- manage tasks (view,delete,update)
- authentication
- login
- register
